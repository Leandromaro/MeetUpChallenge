## Technology used
The challenge was created using the following tools.

  - Java 8
  - Spring Boot
  - Gradle
  - Spring Security
  - Swagger
  - H2 database

### Installation

Follow the next steps to use

```sh
$ git clone
$ cd santanderCallenger
$ ./gradlew build
./gradlew bootRun
```

Default port 8080.
